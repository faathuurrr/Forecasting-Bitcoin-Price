Tentang Aplikasi

Aplikasi ini dibuat untuk memprediksi harga Bitcoin menggunakan metode machine learning berbasis LSTM (Long Short-Term Memory). Data harga diambil secara otomatis dari API Yahoo Finance, sehingga prediksi yang dihasilkan selalu menggunakan data terbaru.

Fitur utama aplikasi:

  • Prediksi harga Bitcoin untuk beberapa hari ke depan
  • Visualisasi grafik harga aktual dan hasil prediksi
  • Pilihan periode prediksi yang fleksibel

Aplikasi ini dikembangkan sebagai bagian dari tugas akhir pada skripsi program studi Sistem Informasi. Model prediksi hanya mendukung Bitcoin, dan hasil prediksi bersifat estimasi, bukan saran investasi.
